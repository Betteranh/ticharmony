# TicHarmony — Guide rapide client

---

## Se connecter

1. Rendez-vous sur le portail TicHarmony
2. Entrez votre **identifiant** et **mot de passe**
3. Sélectionnez votre **profil** (votre nom)

---

## Signaler un problème

1. Cliquez sur **Nouvelle demande**
2. Écrivez un **titre** court — *ex : "Mon imprimante ne fonctionne plus"*
3. Décrivez le problème en détail
4. Choisissez une **catégorie**
5. Vérifiez vos **coordonnées**
6. Cliquez sur **Envoyer**

> Vous recevez un numéro de ticket par confirmation.

---

## Suivre votre demande

Cliquez sur **Mes demandes** dans le menu.

| Statut | Ce que ça veut dire |
|---|---|
| 🔵 **Ouvert** | Votre demande est reçue |
| 🟡 **En cours** | Un technicien s'en occupe |
| 🟢 **Résolu** | Une solution vous attend — **action requise** |
| ⚫ **Clôturé** | Demande terminée |

---

## Quand votre demande est résolue 🟢

Vous recevez une notification. Ouvrez le ticket et choisissez :

- ✔ **Confirmer** → Le problème est bien réglé, merci !
- ↩ **Réouvrir** → Le problème persiste, le technicien est notifié
